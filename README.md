# Erick Mazer's Personal Website

This repository contains the versioned structure of my personal website, with multiple versions representing different iterations and technology stacks.

## Repository Structure

### `v0/` — Original AWS-Hosted Site (Archive)
The original personal website previously hosted on AWS. This folder is **read-only** and serves as a historical archive of the initial implementation. No active development occurs here.

### `v1/` — Current Production Site
The current production website built with **Next.js 16** and **Tailwind CSS v4**, deployed to **Vercel**. This is the active version serving [erickmazer.com](https://erickmazer.com).

### Future Versions
As the website evolves, new versions may appear as `v2/`, `v3/`, etc., each representing significant architectural or technology changes.
